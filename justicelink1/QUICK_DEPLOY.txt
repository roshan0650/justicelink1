🔗 ALTERNATIVE: GITHUB + NETLIFY (2 MINUTES)

Step 1: Push to GitHub
  git init
  git add .
  git commit -m "JusticeLink"
  git remote add origin https://github.com/YOUR_USERNAME/justicelink.git
  git push -u origin main

Step 2: Connect to Netlify
  1. Go to netlify.com
  2. Sign up with GitHub
  3. Click "Add new site" → "Import an existing project"
  4. Select your repository
  5. Set publish directory to: public
  6. Deploy!

Result: Auto-deploys on every push to GitHub
